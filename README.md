# nomhunter-api
NomHunter API service

## Installation

```bash
$ npm install
```

## Running the app
1. Run MongoDB local server 
2. `npm run start:dev`

## Scripts
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# Fix linting
$ npm run lint
```
